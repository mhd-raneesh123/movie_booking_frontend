# Movie Ticket Booking App - Frontend 🍿

This is the React frontend for the Movie Ticket Booking application. It provides a clean, interactive UI for users to browse movies and book tickets, seamlessly communicating with the backend API.

##  Live Demo
**Website:** https://moviebookingfrontend-gold.vercel.app

## 🛠️ Tech Stack
* **Library:** React (Vite)
* **Styling:** CSS
* **Deployment:** Vercel

## 💻 Local Setup

1.Clone the repository:
   ```bash
   git clone https://github.com/mhd-raneesh123/movie_booking_frontend
   ```
2.Install dependencies:
  ```bash
  npm install
  ```
3.Start the development server:
  ```bash
  npm run dev
  ```
4.Open your browser and navigate to http://localhost:5173 (or http://localhost:3000).
